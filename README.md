## 矩阵加法：
   首先两个矩阵会复制至形状一至再进行逐元素相加

## 求转置矩阵：
   矩阵沿左上角到右下角对角线翻转

## 什么是矩阵的broadcasting(广播)：
   两个数组进行运算，前提是两个数组的形状一致，当形状不一致时，较小的矩阵通过复制的方式可以放大至与较大的数组一致的形状的过程叫做广播，如果无法放大至一样的形状那么为不可被广播，此时这两个数组无法进行计算
